# مدیریت Exception و پردازش ورودی

## موضوع
برنامه Console که لیستی از رشته‌ها را به عدد صحیح تبدیل کرده و جمع آن‌ها را محاسبه می‌کند.

## الزامات
1. استفاده از `int.TryParse` برای تبدیل امن
2. **رد کردن** ورودی‌های نامعتبر (بدون crash)
3. **لاگ خطا** برای هر ورودی نامعتبر
4. نمایش **جمع اعداد معتبر** در پایان
5. برنامه نباید با ورودی بد crash کند

## ورودی نمونه
```
["10", "abc", "25", "", "3.5", "7"]
```
خروجی مورد انتظار: جمع `42` (10 + 25 + 7) و لاگ برای موارد نامعتبر.

## فنی
- پروژه **.NET 9** Console
- قابل build و اجرا

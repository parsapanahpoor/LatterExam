# پردازش موازی فایل‌ها در Console App با .NET 9

## اطلاعات آزمون
- **دسته‌بندی:** Concurrency-Scalability
- **نوع پروژه:** Console App (.NET 9)

## شرح مسئله
یک Console App با .NET 9 باید فایل‌های متنی را به‌صورت **موازی** پردازش کند. برنامه باید:

1. پوشه `TestFiles` را ایجاد کند (در صورت نبود)
2. **۲۰ فایل** متنی (`File0.txt` تا `File19.txt`) با محتوای اولیه بسازد
3. همه فایل‌ها را **همزمان** بخواند، محتوا را به حروف بزرگ تبدیل کند و دوباره بنویسد

الگوی صحیح پردازش موازی فایل‌ها با `Task.WhenAll` است.

## چالش‌های اصلی
- ایجاد و مدیریت فایل‌های تست
- پردازش موازی با `Task.WhenAll` بدون بلاک کردن thread اصلی
- استفاده از `ReadAllTextAsync` و `WriteAllTextAsync`

## الزامات و محدودیت‌ها
- فریم‌ورک: **.NET 9**
- بدون وابستگی خارجی
- بدون cache
- قابل build با `dotnet build`

## راهنمای شروع
```bash
dotnet build
dotnet run
```

پس از اجرا، فایل‌های داخل `TestFiles` باید به uppercase تبدیل شده باشند.

# سامانه نظرسنجی آنلاین

## موضوع
طراحی و پیاده‌سازی یک Web API برای سامانه نظرسنجی آنلاین با قابلیت‌های زیر.

## الزامات

### عملکردی
1. **ایجاد نظرسنجی** — ادمین بتواند سوال و گزینه‌های پاسخ را تعریف کند.
2. **رأی‌دهی** — کاربران احراز هویت‌شده بتواند به نظرسنجی رأی دهند (هر کاربر فقط یک بار).
3. **نتایج لحظه‌ای** — نمایش تعداد آرا برای هر گزینه به‌صورت real-time (یا نزدیک به real-time).

### غیرعملکردی
- **احراز هویت JWT** — پیاده‌سازی stub برای توکن JWT.
- **کنترل دسترسی مبتنی بر نقش** — نقش‌های `Admin`، `User`، `Analyst`:
  - `Admin`: ایجاد و مدیریت نظرسنجی
  - `User`: رأی‌دهی
  - `Analyst`: مشاهده نتایج (بدون رأی‌دهی)
- **ذخیره‌سازی رمزنگاری‌شده** — مفهوم رمزنگاری برای داده‌های حساس (مثلاً شناسه رأی‌دهنده).

### معماری
- **Clean Architecture** — لایه‌های Domain، Application، Infrastructure، API.
- پروژه باید با **.NET 9** ساخته و اجرا شود.

## خروجی مورد انتظار
- پروژه Web API قابل build
- endpointهای REST برای عملیات اصلی
- جداسازی واضح لایه‌ها
